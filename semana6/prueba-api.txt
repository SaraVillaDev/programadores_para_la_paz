Se envió una petición POST a http://localhost:3000/registro con el siguiente JSON:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor respondió con Status 200 OK y el siguiente JSON:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto confirma que el servidor recibe correctamente los datos enviados en el
cuerpo de la petición y los devuelve organizados en formato JSON.
